# Credit Card Form

A recreation of a standard credit card form found on eCommerce checkout pages

## Get Started Fast!

### Requirements

```
Node v8.11.1+
npm 5.6.0+
```

### Installation

```
git clone
npm install
npm start | npm run build
```

### Run tests

```
npm run test
```

### Run Airbnb Linter

```
npm run lint
npm run lint:fix
```

## Project Supported By

* Create-react-app
* Node.js | Express.js 
* MongoDB | Mongoose
* Airbnb Linter
* Axios
